Changes in calculation:
Decimal numbers has changed from float to Decimal. 
It's known that 1.1 + 2.2 = is evaluated to3.3000000000000003 in python float.
With Decimal, Decimal('1.1') + Decimal('2.2') is evaluated to Decimal('3.3').
The tiebreak structure has a new element precision, that shows the number of digits after decimal point.

Program:
New switches:
-p : tournament has predefined pairing
-s : tournament is a swiss tournament
-r : sort in rank order (in combination with -d) 
Program returns %errorlevel% 0 if everything is OK
Program returns %errorlevel% 1 with -c switch and rank order does not match
Program returns %errorlevel% >=500 on error

Internal structure:
The sections playerSection and teamSection are replaced by
gameList, matchList, gameScoreSystem, matchScoreSystem and competitors.
competitors are players for individual tournaments and teams for team tournaments.

In return struture
result is changed to tieBreakResult

New program: pairingchecker.py
python pairingchecker.py [-i <input> ] [-o <output> ] [-c] [-a] [-p] [-d T] [-n <round>]
-c = Check mode 
-a - analyze current file 
-p - show pairing 
-d T - formatted text output
-n round - Ruund number to analyze 

python pairingchecker.py -i <input> -d T -p
will give pairing of bext round

python pairingchecker.py -i <input> -d T -c
will check all round in the input file

python pairingchecker.py -i <input> -d T -c -a -p -n 8
will print details albout the pairing in round 8
